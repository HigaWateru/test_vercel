# YÊU CẦU PHÂN TÍCH

Hãy đọc toàn bộ source code trước khi sửa.

Sau đó:

## 1. Phân tích

Liệt kê tất cả vấn đề SEO hiện tại.

Ví dụ:

* thiếu title
* thiếu meta description
* thiếu canonical
* thiếu robots
* thiếu sitemap
* heading sai
* duplicate title
* duplicate description
* hình ảnh không có alt
* URL chưa thân thiện
* thiếu Open Graph
* thiếu Twitter Card
* không có structured data
* render chậm
* JavaScript blocking
* bundle quá lớn
* không lazy load
* không preload font
* font render blocking
* ảnh chưa nén
* CLS cao
* LCP cao
* FCP cao
* INP chưa tối ưu

Đánh giá mức độ:

Critical

High

Medium

Low
